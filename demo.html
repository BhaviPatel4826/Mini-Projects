<a> - Anchor Tag (Link)

The <a> tag defines a hyperlink. You can use various attributes to control its behavior.

  • href: The URL the link goes to
  • target: Where to open the link (_blank, _self, _parent, _top)
  • rel: Relationship between current and linked document (e.g., "nofollow", "noopener")
  • download: Triggers file download
  • type: MIME type of the target document (rare)

Examples:

<a href="https://example.com">Basic Link</a>

Basic Link

<a href="https://example.com" target="_blank">Open in new tab</a>

Open in new tab

<a href="file.pdf" download>Download File</a>

Download File

<a href="https://example.com" rel="nofollow">No Follow Link</a>

No Follow Link

<form> Tag – HTML Forms

HTML forms collect user input. The <form> tag wraps form controls.

Key attributes:

  • action – Where to send the form data (URL)
  • method – HTTP method: get or post
  • autocomplete – Enable/disable browser autocomplete
  • target – Where to open result (e.g. _blank)
